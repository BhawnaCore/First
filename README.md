# Gothic Tic-Tac-Toe

A polished, dark-themed Tic-Tac-Toe experience built with React, TypeScript, and Vite. The game features a medieval gothic aesthetic, move history, and animated win/draw states.

## Features
- Gothic-inspired UI with animated candles and decorative details
- Turn-based gameplay with move history
- Win and draw overlays
- Responsive layout for desktop and smaller screens

## Getting started

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

## Project structure
- src/App.tsx — app entry component
- src/Gothic.jsx — main game UI and logic
- src/index.css — global styles
- public/favicon.svg — project favicon
